TT-NPE Installation Manual

Install tt-npe, run the first workload, and understand the first input and output files you should expect from a clean setup.

Upstream repo
tenstorrent/tt-npe
Validated commit
09d56e9bd91408d449042315d61f91a59c6cbbca
First required input
A workload JSON passed with -w
First success check
Run the bundled workload JSON

Host Requirements

git
cmake
ninja
clang-17
python3

Clone And Build

git clone https://github.com/tenstorrent/tt-npe.git
cd tt-npe
./build-npe.sh

Optional Shell Setup

source ENV_SETUP

Required Input For The First Run

The first run needs a workload JSON file passed with the required -w argument.

Verify The Install

source ENV_SETUP
tt_npe.py -w tt_npe/workload/example_wl.json

Direct invocation also works:

python3 install/bin/tt_npe.py -w tt_npe/workload/example_wl.json

Optional Detailed Output

tt_npe.py -w tt_npe/workload/example_wl.json -e

Adding -e writes npe_timeline.json by default.
